# About


This is bootstrapping part of Trusted Bootstrap project. It is based on [live-bootstrap](https://github.com/fosslinux/live-bootstrap). We're using live-bootstrap for building Linux kernel and dependencies to use in the compiler verification system from minimal binary seeds, in a Trusted way.
